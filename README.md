# Lizzy Translate - Language Translation Tool
**CodeAlpha AI Internship - Task 1**

A simple web app that translates text between 9 languages using translation API logic.

**Features:**
- User can enter text + select source & target language
- Supports 9 languages: Hausa, Yoruba, English, Igbo, Swahili, Zulu, Arabic, Spanish, French  
- Displays translated text clearly on screen
- Mobile-friendly UI for small phone screens
- Optional: Copy button + voice/text-to-speech ready

**Tech used:** HTML, CSS, JavaScript 
**How to run:** Open `index.html` in browser
